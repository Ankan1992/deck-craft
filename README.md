# DeckCraft AI — Professional Presentation Builder

Create stunning, professional presentations in seconds. Upload files or chat to build contextual decks for consulting, tech, business, VC pitches, investment banking, academic, and more.

## Features

- **Chat-Based Creation**: Describe your presentation in natural language
- **File Upload**: Support for .xlsx, .csv, .docx, .txt, .pptx, .json
- **15+ Professional Templates**: Industry-specific designs (McKinsey Blue, Startup Pitch, Tech Dark, BCG Green, IB Classic, etc.)
- **8 Industry Profiles**: Consulting, Tech, Business, VC Pitch, Investment Banking, Academic, Product, Generic
- **Smart Keyword Highlighting**: Auto-bold industry keywords in slides
- **Slide Editor**: Edit individual slides before downloading
- **Live Preview**: Full slide preview before export
- **PPTX Export**: Professional PowerPoint files ready for presentation

## Tech Stack

- **Next.js 14** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **PptxGenJS** for PPTX generation
- **Mammoth** for DOCX parsing
- **SheetJS** for Excel/CSV parsing

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
